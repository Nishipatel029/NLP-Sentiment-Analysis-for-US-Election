# NLP-Sentiment-Analysis-for-US-Election
Dataset: https://www.kaggle.com/datasets/manchunhui/us-election-2020-tweets


## Objective
The objective of this project is to analyze public sentiment during the US elections using Natural Language Processing (NLP) techniques.

- Python
- Pandas
- NLP techniques
- Matplotlib / Seaborn

## Process
- Data cleaning and preprocessing
- Text processing (tokenization, stopword removal)
- Sentiment analysis (positive, negative, neutral classification)
- Data visualization of sentiment trends

## Key Insights
- Identified overall public sentiment towards candidates
- Observed sentiment distribution across different categories
- Highlighted trends in public opinion during the election period

## Conclusion
This project demonstrates how NLP can be used to extract meaningful insights from textual data and support data-driven decision-making in political analysis.
